## Usage

Clone the repository and push it to Heroku, then set the following config variables:
	- `SLACK_API_KEY`
	- `SLACK_CHANNEL`
	- `LAT`
	- `LNG`
	- `CATCH_THRESHOLD`
	- `INTERVAL_MS`

Don't set the INTERVAL_MS too high - I use one 60000 (one minute).
